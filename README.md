# Parrot Pot Home Assistant Integration

## Overview

The Parrot Pot home assistant integration allows you to easily monitor your Parrot Pot from your home automation system. This integration provides access to moisture levels, watering level, and more.

## Features
- battery
- Temperature
- Moisture
- Light
- Water Level

## Installation
1) Copy the parrotpot folder to /custom_components/ of your HA custom_components directory via SSH or SMB
2) Restart Home Assistant
3) Add integration via Settings → Devices & Services → Add Integration → Parrot Pot

## Usage
Once set up, you can access your Parrot Pot directly from your Home Assistant dashboard, where you can view stats of your pot.

## Troubleshooting
- Ensure that your Parrot Pot is not connected to your smartphone.
- Check Home Assistant logs for any errors related to the integration.
